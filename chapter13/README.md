## Twitter微博

### 稍微长一点的API组合应用示例

#### 混合 Twitter API 应用的 4 个命令

命令|描述
----|---
search|采用最近匹配优先的方法在 Twitter 上进行搜索。这是未经验证的调用(应用中只有这一个)，意味着任何人都能调用 
verify_credentials|判断已验证用户的身份是否合法
user_timeline|获取已验证用户最新的推文
update_status|更新已验证用户的状态，也就是说发布一条新的推文
